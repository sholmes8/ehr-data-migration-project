# EHR Data Migration & Workflow Optimization (NextGen)

## Overview
Simulated the transition of a healthcare practice from paper-based records to a structured NextGen EHR system.

## Key Features
- Designed relational database (patients, providers, appointments)
- Modeled data migration from paper → digital records
- Developed SQL queries for patient trends and provider workload
- Built dashboard to visualize operational insights

## Tech Stack
SQL | Data Modeling | CSV | Relational Databases

## SDLC Approach

**Planning**
- Identified challenges in paper-to-digital healthcare transitions
- Defined key entities and workflows

**Design**
- Designed relational database schema
- Structured data for migration simulation

**Development**
- Created datasets and SQL queries
- Modeled relationships between entities

**Testing**
- Validated data consistency and query outputs

**Deployment**
- Hosted project on GitHub
